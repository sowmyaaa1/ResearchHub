# Hedera Wallet Connect Project

A Next.js application for connecting Hedera wallets and managing user profiles.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Features

- Hedera wallet connection
- User authentication with Supabase
- Profile management

## Tech Stack

- Next.js 14
- TypeScript
- Supabase
- Tailwind CSS
